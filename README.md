# BLINKIT-SALES-DATA-ANALYSIS
**Project Overview**

This project is based on analyzing grocery sales data from BlinkIT using Microsoft Excel. The main goal of this project is to understand sales performance, customer preferences, and outlet insights.

**What I Did in This Project**
- Cleaned and organized raw data in Excel
- Analyzed different product categories and their sales
- Studied outlet performance based on location, size, and type
- Evaluated item attributes like fat content, visibility, and weight
- Used Excel tools like:
- Pivot Tables
- Charts & Graphs
- Filters and Sorting
**Key Insights**
- Identified which item types generate the highest sales
- Compared sales across different outlet locations and sizes
- Observed how item visibility and weight affect sales
- Analyzed customer ratings for products
**Tools Used**
Microsoft Excel
Dataset Features

**The dataset includes:**

- Item details (type, fat content, weight)
- Outlet information (location, size, type, establishment year)
- Sales data and ratings
  
**Conclusion**
This project helped in understanding how data analysis can be used to make better business decisions in the retail industry. It also improved my skills in Excel data analysis and visualization.
